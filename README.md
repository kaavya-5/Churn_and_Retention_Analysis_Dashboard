📊 Customer Churn & Retention Analysis Dashboard

🚀 Project Overview

This project focuses on analyzing customer data for a subscription-based business to identify churn patterns, understand retention drivers, and evaluate customer lifetime trends.

The goal is to transform raw customer data into actionable insights that help reduce churn and improve long-term customer engagement.

🎯 Objectives

Identify churn patterns and high-risk customer segments

Perform cohort analysis to track retention over time

Analyze customer lifetime value (CLV) trends

Discover key factors influencing customer retention

Provide data-driven recommendations to reduce churn

🛠️ Tools & Technologies Used

Power BI / Excel / Python (Pandas, Matplotlib, Seaborn)

Data Cleaning & Preprocessing

Cohort Analysis Techniques

Retention Metrics & KPIs

📊 Key Metrics Analyzed

Churn Rate (%)

Retention Rate (%)

Customer Lifetime Value (CLV)

Monthly Active Users (MAU)

Average Subscription Duration

📈 Dashboard Features

📌 Churn rate trend over time

📌 Cohort retention heatmap

📌 Customer lifetime value analysis

📌 Segmentation by demographics / plan type

📌 Revenue vs churn comparison

📌 Interactive filters for deep insights

🔍 Key Insights

Higher churn observed in early subscription months

Long-term customers show significantly higher lifetime value

Certain subscription plans have better retention rates

Engagement levels strongly influence customer retention

💡 Retention Drivers Identified

Customer engagement frequency

Subscription plan value

Customer support experience

Pricing and offers
